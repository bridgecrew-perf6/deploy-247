## Introduction

*issuerail* is the issue tracker written in Rust and Vue.



## Prerequisites

- Rust >= 1.59.0
- Vue >= 3.0.0



## Run

- **Server**

```bash
git clone https://github.com/issuerail/server.git --depth 1

cd server
./server.sh build
./server.sh run
```



- **Web**

```bash
git clone https://github.com/issuerail/web.git --depth 1

cd web
./web.sh build
./web.sh run
```



## Docker

```bash
./compose.sh build
./compose.sh run
```
